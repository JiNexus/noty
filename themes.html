NOTY is a notification plugin that makes it easy to create alert - success - error - warning - information - confirmation messages as an alternative the standard alert dialog.

Theme Library

NOTY comes with 6 themes; relax, mint, sunset, metroui, semanticui, bootstrap-v3, bootstrap-v4 and the default is mint.

If you want to submit your theme you can create a pull request on Github.
